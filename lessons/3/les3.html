Блочные и строчные элементы

В CSS мы, говоря упрощённо, имеем два типа элементов — блочные и строчные. Эти характеристики относятся к поведению блоков в контексте потока страницы и относительно других блоков на странице.

Блочные элементы

Блочные элементы занимают всю ширину родительского контейнера и всегда начинаются с новой строки. Они используются для создания структуры страницы.

Примеры блочных элементов

  • div
  • p
  • h1 .. h6
  • ul и ol
  • form
  • table

Свойства блочных элементов

  • Занимают всю доступную ширину родительского элемента
  • Начинаются с новой строки.
  • Могут иметь ширину и высоту.
  • Поддерживают все типы внутренних и внешних отступов (padding и margin).

Строчные элементы

Строчные элементы занимают только необходимую для их контента ширину и не начинают новую строку. Они используются для стилизации частей текста внутри блочных элементов.

Примеры строчных элементов

  • span
  • a
  • strong .. h6
  • em и ol
  • img
  • input

Свойства строчных элементов

  • Занимают только необходимую для их контента ширину.
  • Не начинают новую строку.
  • Не могут иметь ширину и высоту.
  • Поддерживают только внутренние отступы (padding), но не внешние отступы (margin).

Изменение поведения элементов с помощью CSS

С помощью CSS мы можем изменить поведение элементов с помощью свойства display.
  • display: block; — делает элемент блочным.
  • display: inline; — делает элемент строчным.
  • display: inline-block; — делает элемент строчным-блочным (сочетает в себе свойства как блочных, так и строчных элементов. Он располагается в одной строке с другими элементами, но при этом можно задавать ширину и высоту).
